# Keithley CLI Analysis Notebook

This repository contains a Jupyter notebook for analyzing data from Keithley CLI measurements. The notebook includes code for loading, processing, and visualizing the data, making it easier to understand and interpret the results.

## Getting Started

Launch the notebook using Binder by clicking the badge above or by visiting this [link](https://mybinder.org/v2/gh/JYU-RADEF/keithley-cli-analysis-notebook/HEAD?urlpath=%2Fdoc%2Ftree%2FKeithleyCLIAnalysis.ipynb).
